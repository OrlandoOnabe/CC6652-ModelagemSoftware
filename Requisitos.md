## Requisitos Funcionais

    RF1: O sistema deve permitir que os clientes verifiquem o catálogo.
    RF2: O sistema deve permitir que os clientes escolham uma sessão de um filme.
    RF3: O sistema deve permitir que os clientes reservem assentos e ao fazer isso recebam ingresso físicos ou digitais
    RF4: O sistema deve permitir que clientes comprem meia-entradas.
    RF5: O sistema deve dar um ingresso grátis no mês de aniversário dos clientes que fazem parte do programa de fidelidade.
    RF6: O sistema deve permitir que o administrador selecione filmes da lista oficial da ANCINE para disponibilizá-los nas salas de cinema.
    RF7: O sistema deve permitir que os administradores gerenciem filmes.
    RF8: O sistema deve permitir que os administradores cadastrem filmes.
    RF9: O sistema deve permitir que os administradores gerenciem sessões.
    RF10: O sistema deve permitir que os administradores cadastrem sessões.
    RF11: O sistema deve permitir que os clientes no programa de fidelidade comprem ingressos 2 dias antes das pessoas que não fazem parte do programa.
    RF12: O sistema deve permitir que os clientes no programa de fidelidade escolham os melhores assentos, se comprarem com 2 dias de antecedência conforme o RF11.

## Requisitos não funcionais:

    RNF1 - O sistema não deve permitir que assentos sejam reservados em duplicidade por sessão
    RNF2 - O sistema deve garantir o controle de disponibilidade de forma confiável
    RNF3 - O sistema deve controlar a disponibilidade de assentos em tempo real
    RNF4 - O sistema deve permitir a compra de meia-entrada somente para: estudantes devidamente identificados, pessoas maiores de 65 anos, professores
    RNF5 - O sistema deve gerar relatórios automáticos de venda e renda após o término de cada sessão
    RNF6 - O sistema deve registrar quais filmes estão disponibilizados por sala de acordo com a lista oficial da ANCINE
    RNF7 - O administrador só pode selecionar filmes da lista oficial da ANCINE
    RNF8 - O programa de fidelidade deve aplicar de forma correta seus benefícios
    RNF9 - O sistema deve se integrar com um serviço da ANCIEN para obter a lista de filmes disponíveis e disponibilizá-los por sala
    RNF10 - O sistema deve manter histórico de reservas e vendas para auditoria.
    RNF11 - O sistema deve registrar a quantidade de ingressos de meia-entrada vendidos por
    categoria (estudantes, maiores de 65 anos, professores)
    RNF12 - O sistema deve registrar, para cada sessão, a quantidade de ingressos vendidos e a renda obtida
